# AI-Powered E-Commerce Command Center

An AI-powered business command center that monitors e-commerce operations in real time, generates alerts automatically, explains business issues using AI, and allows users to interact with data using natural language.

## Features

* Real-time KPI monitoring
* AI-generated alerts
* Root-cause analysis using LLMs
* Conversational analytics
* Forecasting
* Teams and Email notifications
* Single-page executive dashboard

## Tech Stack

* Power BI
* Python
* FastAPI
* Supabase
* Azure OpenAI / Google AI Studio
* GitHub
* Microsoft Fabric (optional)
